Develop Mobile App (TK2) with Expo -- Material Design

Node JS: v18.19.1 (LTS)

Initialize: npx create-expo-app@latest loginMaterial

NPM Dependency: 
- npx expo install @mui/material @emotion/react @emotion/styled @mui/icons-material @heroicons/react


Run Project: npx expo start -c

Build APK: https://docs.expo.dev/build-reference/apk/
npm install --global expo-cli eas-cli



Refs:
https://docs.expo.dev/tutorial/create-your-first-app/
